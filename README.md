# Requirements 
 -   Linux server (recommended)
 -   Python >= 3.6
 -   PostgreSQL
 -   MongoDB 
# Features
 - Login with Google account
 - Login with QR code
 - Making online sweepstakes
 - Follow the sweepstakes on Instagram
 - Add raffles through Instagram
 - Search for raffles in the application
 - Multi-language support

# Related projects
